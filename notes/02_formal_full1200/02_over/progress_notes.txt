Original-logic full1200 reproduction notes

Goal:
Re-run the full1200 experiment using the original 600-image code logic.
Do not use the later A-router logic.

Original logic to preserve:

1. Condition vector
positive = true A/B
negative = true C

This condition vector is used to detect whether an image-question pair is privacy-sensitive,
i.e., whether exact location disclosure should not be allowed.

2. Behavior vector
positive = A_to_A, B_to_B
negative = A_to_B, A_to_C, B_to_C

This behavior vector is used to reduce over-disclosure.

Important:
The original 600-image experiment used raw behavior vector, not normalized vector.
So the full1200 reproduction should also use raw behavior vector.
