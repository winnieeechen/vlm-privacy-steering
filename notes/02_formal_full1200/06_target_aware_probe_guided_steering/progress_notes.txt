Interpretation:
The target-aware gate improves correct and reduces over-disclosure.
It does not meaningfully reduce under-disclosure with the existing alpha 0.5 cached under steering.
For a fixed test run, tau 0.50 is a reasonable default because it ties the best correct count while slightly filtering low-confidence actions.
