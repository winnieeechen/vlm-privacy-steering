Dual-additive original-logic full1200 experiment

Goal:
Run true combined activation steering with both over and under vectors added in the same forward pass.

This is different from dual-router cached combine.

Previous dual-router:
If over gate is true, use over-steered output.
Else if under gate is true, use under-steered output.
Else use base output.

Dual-additive formula:
h'_{l,t} = h_{l,t}
          + g_privacy * alpha_over * v_privacy,l
          + g_allowed * alpha_under * v_utility,l

Where:
g_privacy = 1 if over condition score >= over threshold, else 0
g_allowed = 1 if under condition score >= under threshold, else 0

Current thresholds selected from val:
over threshold = -0.03
under threshold = 0.04

Current alpha/layer setting:

alpha_over = 0.5
alpha_under = 0.5
