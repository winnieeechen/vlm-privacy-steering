Vector type:

Goal:
Reduce over-disclosure using the same high-level logic as the original
full1200 over-disclosure experiment, but replace the raw mean-difference
vectors with low-rank Fisher/PCA vectors.

Interpretation:
The condition vector does separate C from A/B in the expected direction, but the
score range is very narrow. Because the existing sweep thresholds were inherited
from the mean-difference experiment, most thresholds are too low for this new
score scale and therefore turn the gate on for every example.

Conclusion
----------
The fisher_pca over vectors are directionally useful but weaker than the
original mean-difference over vectors.

They improve over the base model:
Correct +3
Over-disclosure -3

But they do not match the original over-disclosure steering result:
Correct 86 vs 94
Over 37 vs 22

For reporting, this should be treated as a negative or exploratory result rather
than the main method.


Recommended next checks
-----------------------
1. Reuse the cached activations to generate alternative vector methods:
   - ensemble
   - pca_residual
   - pca_projected

2. Use a tighter threshold sweep around the actual fisher_pca score scale:
   -0.020, -0.0175, -0.015, -0.0125, -0.010, -0.0075,
   -0.005, -0.0025, 0.000, 0.0025, 0.005

3. Do not run test for this exact fisher_pca over setting unless needed for a
   completeness table. The val result is already clearly worse than the
   original mean-difference over result.
